# Error Taxonomy

## Critical convention errors

- unsupported market convention
- basis orientation error
- product scope promotion error
- incorrect quantity convention
- period grammar error
- unsupported vol surface treated as supported

## Data errors

- missing mark
- stale mark
- missing timestamp
- unknown product
- unknown gas location
- missing delivery window

## Calculation errors

- spread sign reversal
- ATC naive averaging
- wrong gas contract sizing
- strip weighting error
- PnL bridge does not sum

## Interpretation errors

- inference presented as fact
- unsupported causal claim
- unlabeled default
