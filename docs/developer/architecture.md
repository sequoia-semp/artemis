# Developer Architecture

Artemis is the product identity. `pga` remains the package command family for existing
deterministic analytics, and `artemis` is the user-facing alias for agent modes and manifests.

The current slice adds:

- `artemis.yaml` central configuration
- Analyst and Development mode contracts
- optional provider and coding-backend descriptors
- tool and permission registries
- data-source descriptors
- schema-backed Analyst views
- skill and view manifest validation

No locked market conventions changed.
