# TUI Roadmap

The full Textual/TUI interface is deferred. The current slice adds renderer placeholders and a
schema-backed view engine so future panels can consume deterministic view payloads.

Target panels:

- Chat/task input
- Views and dashboards navigation
- Main table/chart/report panel
- Evidence and lineage panel
- Missing-input and data-quality panel
- Tool-trace and validation panel
