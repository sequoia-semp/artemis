# Knowledge Base

The Artemis knowledge base is a deterministic scaffold for reviewed explanations, examples, and source notes.

KB entries are advisory unless linked to at least one canonical artifact:

- locked convention
- registry entry
- schema
- source documentation
- deterministic test

KB entries cannot override:

- `docs/CONVENTIONS_LOCKED_v0.1.md`
- `domain/`
- `registries/`
- `schemas/`
- tested analytics code

Do not add market convention as fact unless it is already locked, registered, schema-backed, source-backed, or covered by deterministic tests.
