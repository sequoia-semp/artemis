# Skill: ATC and Strip Decomposition

## Purpose
Expand ATC and strip periods into component exposures.

## Rules
1. ATC = equal MW peak + equal MW offpeak.
2. Apply period parser before valuation.
3. Value components independently.
4. Display ATC blended price as hour-weighted average.
5. BALMO excludes next-day product.

## Required output
- component exposures
- derived MWh
- blended price
- lineage
